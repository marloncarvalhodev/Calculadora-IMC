# Calculadora de IMC

Aplicação web simples para calcular o Índice de Massa Corporal (IMC) 
e classificar o resultado automaticamente.

## Funcionalidades
- Cálculo do IMC a partir de peso e altura informados pelo usuário
- Classificação automática (abaixo do peso, peso normal, sobrepeso, obesidade)

## Tecnologias
- HTML
- CSS
- JavaScript

## Como usar
Clone o repositório e abra o arquivo index.html no navegador.
